Tích hợp thanh toán qua cổng thanh toán 9PAY cho Woocommerce

== Description ==

Tích hợp thanh toán quét mã một cách đơn giản nhất
Liên hệ với hệ thống chăm sóc khách hàng của 9PAY để được hỗ trợ kết nối https://9pay.vn/contact

== Installation ==

1. Cài đặt Plugin như bình thường
2. Kích hoạt Plugin, nhớ cài đặt plugin Woocommerce trước
3. Vào Woocommerce -> Setting -> Payment
4. Chọn các cổng thanh toán 9pay và thực hiện kết nối
